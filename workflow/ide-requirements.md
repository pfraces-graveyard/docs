IDE requirements
================

What I expect from an IDE?
--------------------------

### File capabilities

*   reindent
*   syntax highlight
*   line numbers
*   wrap / unwrap line content into visible area
*   multicursor

### Project integration

*   file browser
*   tabs and layouts
*   embeded shells
*   find / replace at project level

### Tools integration

*   autocomplete at project level
*   linter integration
*   debug with breakpoints

### Productivity

*   key bindings
*   snippets
