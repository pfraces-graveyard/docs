# HJKL

*   `H/L`: Scroll Left/Rigt
*   `J/K`: Scroll Down/Up
*   `shift + H/L`: Back/Forth in history
*   `shift + J/K`: Back/Forth in tabs

# tabs

*   `t`: new
*   `yt`: duplicate
*   `x`: close
*   `shift + x`: restore

# url clipboard

*   `yy`: copy current url to the clipboard
*   `p`: open url from clipboard in current tab
*   `shift + p`: open url from clipboard in new tab
